- 功能概述:包含一言,必应壁纸,天气,听力播放器,倒计日(倒计时)

## 如何使用

> 希沃一体机请确保关闭冰点还原

下载最新版：<https://github.com/Cokeenet/CokeeDP/releases/latest> 后，解压文件，找到CokeeDP.scr，右键选择”安装“即可
